# Keyboard Shortcuts

Work faster with keyboard shortcuts.

## Global Shortcuts

- **Ctrl+Shift+L**: Quick launch workspace
- **Esc**: Close dialogs
- **Ctrl+F**: Search workspaces
- **Ctrl+,**: Open settings

## Workspace Management

- **Ctrl+N**: New workspace
- **Ctrl+D**: Duplicate workspace
- **Ctrl+Delete**: Delete workspace
- **Ctrl+Enter**: Rename workspace

## Navigation

- **↑ / ↓**: Navigate workspace list
- **Enter**: Launch selected workspace
- **Tab**: Focus next element
- **Shift+Tab**: Focus previous element

## Pro Tips

- **Customize shortcuts**: Settings → Hotkeys
- **Avoid conflicts**: Check system shortcuts first
- **Learn gradually**: Start with 3-4 shortcuts
- **Practice**: Use them daily to build muscle memory

All shortcuts are configurable in Settings.
